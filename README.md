## How To Run

```bash
1. Clone the repository:
  git clone <repository-url>
  cd <repository-folder>

2. Install npm :
  $ npm install

3. create .env on root directory and copy field from .env.example

4. Run Application :
  # development
  $ npm run start

  # watch mode
  $ npm run start:dev

  # production mode
  $ npm run start:prod

5. Run Unit Test :
  # unit tests
  $ npm run test

  # e2e tests
  $ npm run test:e2e

  # test coverage
  $ npm run test:cov

```
